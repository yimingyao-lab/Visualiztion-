User Achievement: Comparison of annual budget of categories in the selected year.
Goal: User can find the distribution in each category, multiple selecting can help users to visualize distribution in the difference year. 
    
How: Users can use the keyboard to select different year and draw the heptagon visualization.
Key 1 - 9:  1 - 2014, 2 - 2015, 3 - 2016 etc..
pressed again -> empty the graph
Key r : restart visualization

When the mouse close (nearly) to the categories, the users can see the window to show the amount value of the selected years in detailed.(comparison)
